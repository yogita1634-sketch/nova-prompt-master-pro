# 🎯 NOVA Prompt Master Professional
**Methodology:** P-CTCO (Persona, Context, Task, Constraint, Output)

### 🚀 Overview
Built to standardize high-fidelity prompting, this tool automates the process of aligning LLM outputs with professional intent. It utilizes a structured framework to ensure models adopt a specific persona and adhere to business constraints, reducing hallucinations.

### 🧠 The P-CTCO Framework
This tool guides users through 5 critical stages of model alignment:
1. **Persona Anchoring:** Locks the model into a specific professional role (e.g., Data Scientist, Marketing Director).
2. **Contextual Grounding:** Provides the background scenario to narrow the probability space.
3. **Actionable Tasking:** Direct, clear instructions for the model to execute.
4. **Constraint Setting:** Enforces guardrails such as tone, length, or specific business frameworks (SWOT, OKRs).
5. **Output Specification:** Defines the final structure (Markdown, Table, Executive Summary).
